# DainagonProject

**開発者:** 太田陽道 / 角野俊樹

## プロジェクト概要
このプロジェクトは、フロムスクラッチで「そこそこ使える」将棋AIを製作することを目的としたものです。

DL系将棋AIとして、AlphaZeroのやり方ではない何らかの方法で「そこそこ強い」ものを作ることをゴールとします。

### 利用規約
- 本リポジトリにあるプログラム等の無断転載は禁止します。
- 個人利用の範囲内であれば、どなたでも自由にお使いいただけます。
- モデルデータは半無料です。詳細や入手については [DainagonProject@gmail.com](mailto:DainagonProject@gmail.com) までご連絡ください。

### 現状
モデル制作を太田が、エンジン制作を角野が担当しています。

---

> [!TIP]
> **とにかく強い将棋AIを求めている方へ**
> 本プロジェクトは学習と製作を楽しむためのものです。純粋な強さを求める方は、水匠やdlshogiなどの既存の強力なAIをご利用いただくことをおすすめします。
